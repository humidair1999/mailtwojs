mailtwo.js - A better way to handle "mailto:" links across your site

a jQuery plugin by j. ky marsh

What it is

Mailtwo.js is a jQuery plugin that snatches all your

links and transforms them into styleable modals, allowing your users to easily draft an email or locate you on social networks, rather than simply dumping them into a blank email client.

Additionally, mailtwo.js optionally allows you to use an "encrypted" email address, such as 

fake.guy[at]emailprovider[dot]com

as the mailto: recipient, and it will be properly decoded and placed into the modal at runtime.

How it's used

In its simplest format, simply place a mailto: link somewhere on your webpage, and then toss the mailtwo.js initialization snippet on your page.

You can either initialize mailtwo.js directly on a single mailto: link, or on any DOM element you want! Mailtwo.js will automatically traverse through whatever element you specify, searching for mailto: links and attaching itself to them. 

$("#aLink").mailtwo();
$(".linkList").mailtwo();

Options

// mailtwo.js has 5 customizable properties:

$("#aLink").mailtwo({
    facebook: "", // your facebook username (facebook.com/[username])
    twitter: "", // your twitter username (twitter.com/[username])
    linkedin: "", // your linkedin... thing (linkedin.com/[...thing])
    modalBgClass: "", // CSS class for the modal's background
    modalClass: "" // CSS class for the modal itself
});

// all 5 of them are optional, and the last 2 have defaults:

modalBgClass: "mailtwoBackground",
modalClass: "mailtwoModal"

Some notes

  • Mailtwo.js includes a .css file that the modal and background rely on for styling. Don't forget to include it! (If you override the default CSS classes, of course, styling is then up to you.)
  • Speaking of which: by all means, customize the default .css file! I'm not a designer by any means, so the default is a bit hideous.
  • If you initialize multiple instances of mailtwo.js, it's best if they all share the same CSS classes. If you instantiate multiple instances, all with different classes, bad things will happen, I promise.
  • The social media icon font used by default is the wonderful Socialico. You'll need to download and reference it if you intend to also use it. It's free!
